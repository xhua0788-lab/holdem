# 公平德州扑克 · 朋友联机局

真人对战的德州扑克。洗牌、发牌、比牌全部在**服务器**完成，每个人只收到自己的底牌，
别人偷不到；每局开局公示整副牌的 SHA-256 指纹，结束公开完整牌序，任何人可自行核对公平性。

## 功能
- 创建房间 / 输房号加入，朋友各用各的手机一起玩
- 2 人即可开局，最多 6/8/9 人
- 整局 30/60 分钟或不限时；行动限时 15 秒，可不限次「延时 +15 秒」
- 每手结束 5 秒自动开下一局（可手动点立刻开始）
- 战绩面板：每人带入 / 当前 / 盈亏 + 逐局历史
- 加注推荐：底池 1/3、1/2、2/3、1 池、1.5 池、ALL IN，外加拖动滑杆
- 全员弃牌自动翻完公共牌+亮全部底牌复盘；河牌跟注到底强制全员亮牌
- 掉线保留座位 60 秒可重连；纯积分、无充值入口

## 本地运行（可选，电脑上自测）
```
npm install
npm start
```
然后浏览器打开 http://localhost:3000
